You should have:

💻 Common Git Commands

  1. git init - Initialize a repository
  2. git status - Check repository status
  3. git add . - Stage all files
  4. git commit -m "message" - Commit changes
  5. git log - View commit history
  6. git branch - List branches
  7. git checkout branch-name - Switch branch
  8. git merge branch-name - Merge branches
  9. git clone URL - Clone repository
  10. git push - Push changes
  11. git pull - Pull latest changes

🌟 Why Learn Git?

Git is used by millions of developers because it helps you:

  • Track project history
  • Undo mistakes safely
  • Collaborate with other developers
  • Manage multiple project versions
  • Contribute to Open Source projects
